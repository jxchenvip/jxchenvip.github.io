迭代器模式

在迭代器模式中，通常有一个包含某种数据集合的对象。该数据可能存储在一个复杂数据结构内部，而要提供一种简单的方法能够访问数据结构中每个元素。对象的消费者并不需要知道如果组织数据，所有需要做的就是取出单个数据进行工作。

在迭代器模式中，对象需要提供一个next()方法。依次调用next()必须返回下一下连续的元素。当然，在特定数据结构中，”下一个”所代码的意思是由您来决定的。
