工厂模式

设计工厂模式的目的是为了创建对象。它通常在类或者类的静态方法中实现，具有下列目标：

  • 当创建相似对象时执行重复操作。
  • 在编译时不知道具体类型（类）的情况下，为工厂客户提供一种创建对象的接口
