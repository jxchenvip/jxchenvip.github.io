# 阶乘

在数学上, 一个正整数 `n` 的阶乘 (写作 `n!`), 就是所有小于等于 `n` 的正整数的乘积. 比如:

```
5! = 5 * 4 * 3 * 2 * 1 = 120
```

## 算法

```js
/**
 * @param {*Number} n
 * @return {*Number}
 */
export default function factorial(n) {
  if (n <= 1) return 1;
  return n * arguments.callee(n - 1);
}

```
